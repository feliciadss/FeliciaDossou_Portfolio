Date: Dec 2023

Employee Attrition Factors Analysis (EN)

This project analyzes employee attrition using R for data manipulation, statistical analysis, and visualization. It employs Data Mining techniques like Logistic Regression, Decision Tree Regression, and Association Rule Mining to uncover patterns and factors affecting employee turnover. Data preprocessing methods include data cleaning, transformation, and correlation analysis. Performance evaluation of the models is conducted through accuracy metrics, confusion matrices, and association rule metrics like support, confidence, and lift.

Date: Nov 2023

E-Commerce Sales Optimization Analysis (EN)

This project utilizes data science techniques to analyze e-commerce data, focusing on customer behaviors and transaction patterns. Using Python and libraries like Pandas and Matplotlib, the study cleans and explores a UK-based e-commerce dataset. Machine learning models, including Linear Regression and Gradient Boosting Regressor, are applied for sales prediction. Key findings suggest the importance of product selection and seasonality in sales, with the Gradient Boosting Regressor emerging as the most effective predictive model. The study highlights data science's role in optimizing e-commerce strategies.

Date: Nov 2023

Malaysia's Consumer Price Index Analysis (EN)

This study examines Malaysia's Consumer Price Index (CPI) from 2021 to 2023, using RStudio for data analysis and visualization. The research focuses on the CPI's impact on sectors like 'alcohol_tobacco' and 'food_beverage', and its relation to consumer demand and economic stability. It provides insights for businesses and consumers on market strategies and living expenses.

Air Quality Study in New Zealand in 2021 (FR)
